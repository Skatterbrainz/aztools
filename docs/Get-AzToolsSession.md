# Get-AzToolsSession

## SYNOPSIS
Show Active AZTools context

## SYNTAX

```
Get-AzToolsSession
```

## DESCRIPTION
Show details about current AZTools context, such as TenantID, SubscriptionID,
Subscription Name, ResourceGroup Name and Automation Account Name

## EXAMPLES

### EXAMPLE 1
```
Get-AzToolsSession
```
